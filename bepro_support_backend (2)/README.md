# BePro Support Backend

This is a Spring Boot backend project for BePro Support platform.  

## Features (in-progress)
- User management (with role-based authentication)
- Module and lesson management
- Lesson with questions
- Spring Security with JWT / Basic Auth (private access)

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Lombok

## Getting Started
1. Clone the repository
2. Setup PostgreSQL database `support`
3. Update `application.yaml` with your credentials
4. Run Spring Boot application
